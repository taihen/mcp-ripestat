# mcp-ripestat

`mcp-ripestat` is a Go-based, single-binary, locally run MCP server that fetches and serves data from the RIPEstat Data API.

## Quick Start

### Build

To build the binary, run:

```sh
go build -o mcp-ripestat ./cmd/mcp-ripestat
```

### Run

To run the server:

```sh
./mcp-ripestat
```
